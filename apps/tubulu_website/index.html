04

Designed with
merchants in mind

The Tubulu Admin Portal celebrates freedom and self-sufficiency. Merchant owners control every aspect of their commerce operation from a single dashboard.

From Catalogue management and AI Playbooks to Branch coordinates and Commissions — every module is a click away.

1 / 2

The admin portal's module architecture mirrors Tubulu's backend schema. Each module maps directly to a PostgreSQL table and a set of REST API endpoints in the Node.js backend.

Powered by React + Vite + Tailwind, the portal delivers sub-100ms UI transitions with optimistic updates.
